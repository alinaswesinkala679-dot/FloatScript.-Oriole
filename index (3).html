From a Teenager's Bedroom to Your Machine

FloatScript was born from the frustration of a young programmer tired of unnecessary complexity in modern languages.

Lusungu Luhana (14-Year-Old Developer)

At just 14 years old, Lusungu taught himself C++ by reading documentation and experimenting with compiler design. After becoming frustrated with the verbosity of modern languages, he set out to create FloatScript - a language that keeps the power of low-level programming while making the syntax accessible to everyone.

"I built FloatScript because I wanted a language where lex x = 10 just works—no #include or int main() nonsense. Programming should be about solving problems, not memorizing boilerplate."

Simplicity First

FloatScript eliminates unnecessary syntax while maintaining powerful features:

// Dynamic typing with lex
lex x = 10 // Integer
lex y = "text" // String
lex z = [1,2,3] // Array

vs. C++: std::vector<int> x = {1,2,3};

Power When Needed

Underneath the simple syntax lies serious power:

// Memory management
mem::alloc buffer 1024
mem::write_byte buffer 0 255
mem::free buffer

Full control without the complexity

Learn FloatScript in 60 Seconds

Follow these steps to get started with FloatScript today

1

Install FloatScript

Download the appropriate package for your system:

# Linux/macOS
curl -fsSL https://floatscript.org/install.sh | bash

# Windows
winget install FloatScript.FloatScript
Or download manually from our downloads page
2

Run Your First Program

Create a file called hello.flt with:

std::out "Hello World!"

Then run it with:

floatscript hello.flt
Output: Hello World!
3

Explore the Language

Try these examples to see FloatScript in action:

// Variables
lex name = "Alice"
lex age = 25

// Arrays
lex scores = [90, 85, 95]

// Loop
for _score : scores_ {
  std::out "Score:" + score
}
